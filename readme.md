# ⌨️ Typing Speed Test App

A web-based typing speed test that measures typing speed and accuracy in real time.

🚀 Features
- Real-time Words Per Minute (WPM) calculation  
- Accuracy tracking  
- Mistake detection with visual feedback  
- Random paragraph generation  
- Restart functionality  
- Responsive UI  

🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript
