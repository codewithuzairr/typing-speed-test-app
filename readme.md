Build a **Typing Speed Test Web App** using only **HTML, CSS, and Vanilla JavaScript (no frameworks)**.

### Core Features:

1. Display a random paragraph for typing.
2. Show a typing input area below the paragraph.
3. Start timer automatically when user starts typing.
4. Test duration: 60 seconds.
5. After time ends:

   * Disable input
   * Show results:

     * Words Per Minute (WPM)
     * Accuracy (%)
     * Total mistakes

### Functional Requirements:

* Highlight correct characters in green and incorrect in red while typing.
* Real-time mistake tracking.
* WPM should update in real time.
* Accuracy = (correct characters / total typed characters) * 100
* Restart button to reset test.

### UI Requirements:

* Clean and modern UI
* Centered layout
* Responsive design (mobile + desktop)
* Show:

  * Timer countdown
  * WPM
  * Mistakes count

### Additional Features (Important):

* Generate different random paragraphs each time
* Prevent copy-paste in input
* Smooth animations (cursor or typing feedback optional)

### Code Requirements:

* Use only HTML, CSS, and plain JavaScript
* Separate files: index.html, style.css, script.js
* Code should be clean and well-commented
* No external libraries or frameworks

### Bonus (if possible):

* Dark mode toggle
* Sound effect on typing errors

### Output Format:

* Provide complete working code for all 3 files
* Ensure code runs directly in browser without setup
